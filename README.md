# Vision Badminton Website

This folder contains the organized website project for Vision Badminton.

## Folder structure

- `site/`
  - The live website files used for local preview and handoff.

- `site/pages/`
  - All website pages.

- `site/assets/`
  - Logos, photos, coach images, and other website visual assets.

- `site/css/`
  - Shared website stylesheet.

- `site/js/`
  - Shared website interactions and lead funnel logic.

- `design/`
  - Brand, UI, and visual direction notes for the website.

- `content/`
  - Content planning notes, page summaries, and copy structure references.

- `deployment/`
  - Local preview instructions, launch notes, and deployment preparation docs.
